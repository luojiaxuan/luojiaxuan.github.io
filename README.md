# Jiaxuan Luo Academic Website

Jiaxuan Luo 的个人学术主页，聚焦 streaming multimodal systems、post-training 与高性能推理。站点基于 [al-folio](https://github.com/alshedivat/al-folio) 定制，并通过 GitHub Pages 自动发布。

## 本地预览

```bash
bundle install
bundle exec jekyll serve
```

浏览器访问 `http://127.0.0.1:4000`。

## 内容维护

- 首页简介：`_pages/about.md`
- Selected Work：`_data/work.yml`
- 经历与教育：`_data/experience.yml`、`_data/education.yml`
- News：`_news/`
- Publications：`_bibliography/papers.bib`
- 视觉样式：`_sass/_custom.scss`
